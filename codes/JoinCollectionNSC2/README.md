# JoinCollectionNSC2

### 분석목적
DDA = 선택한 설정변수 중 각자 분석을 맡은 컬럼을 추출하고,
     컬럼 별 데이터 타입과 정보를 확인하여 연속형(int)에서 범주형(object)으로 변경하는 작업을 진행.

EDA = 범주형/연속형에 따라서 그래프를 통해 시각화를하여 목표변수에 영향 여부를 예측해 봄. 

CDA = 목표변수와 설명변수의 관계를 검증하여 p-value값을 통해 수치적으로 영향 여부를 판단함.
